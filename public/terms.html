Terms · Free competition beta

Use the evidence as a tool—not as a guarantee.

Context MRI is free, open-source beta software for task-specific context diagnosis. It can reveal useful evidence, but it does not replace human review, security testing, or production evaluation.

Permission to use

You may use and modify the project under its MIT license. The public demo and local plugin are offered without billing for this competition beta.

Appropriate use

  • Use Context MRI only with content you are authorized to process.
  • Do not rely on a fixture or one task-specific evaluator as proof of broad model safety or performance.
  • Review suggested repairs before applying them and run representative production tests afterward.

Evidence limits

Bundled examples are deterministic fixture replays. File labels mean “harmful, useful, or required for this task under this evaluator,” not universally. Single-file ablation can miss interactions, and fingerprint checks are tamper-evident controls rather than authorization signatures.
